# SmartPi

Offline AI chatbot with a ChatGPT-style GUI.

## Installation

1. Create and activate a Python virtual environment:
\`\`\`bash
python3 -m venv venv
source venv/bin/activate
\`\`\`

2. Install SmartPi:
\`\`\`bash
pip install .
\`\`\`

3. Run the chat interface:
\`\`\`bash
smartpi-chat
\`\`\`

## Custom Responses

Edit \`smartpi/text_model.py\` to add or change phrases and replies.

Enjoy!
